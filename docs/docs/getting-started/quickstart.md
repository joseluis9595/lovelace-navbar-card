# Quick Start Guide

This guide will help you get the Navbar Card up and running in your Home Assistant dashboard in just a few minutes.

## Adding the Card to Your Dashboard

Once you've [installed the Navbar Card](/docs/getting-started/installation), adding it to your dashboard is straightforward:

1. Open your dashboard in **edit mode** (click the three dots menu in the top right and select "Edit Dashboard")
2. Click the **"Add Card"** button (+ icon) at the bottom right
3. Search for **"Navbar Card"** in the card picker
4. Select it to add to your dashboard

![Add navbar-card to your dashboard](/img/quickstart/navbar-card_tutorial_1-add-card.gif)

:::tip Auto Configuration
The card comes with a default configuration that includes common routes (Home, Settings, Map), so you can start using it right away!
:::
